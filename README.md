API testing
=================


API implementation with Node, Express, MongoDb and Vercel.

URL: https://node-api-users.vercel.app/api

Endpoints:

* GET `/api/users`
* POST `/api/users`
* GET `/api/users/:user_id`
* PUT `/api/users/:user_id`
* PATCH `/api/users/:user_id`
* DELETE `/api/users/:user_id`
